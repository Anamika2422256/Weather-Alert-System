# Weather Alert System

## Description
This project fetches real-time weather data using the OpenWeatherMap API and sends email alerts using SMTP.

## Features
- Retrieves current weather data using OpenWeatherMap API.
- Checks if temperature exceeds 35°C or rain is predicted.
- Sends email notifications automatically.

## Technologies Used
- Python
- OpenWeatherMap API
- SMTP
- Gmail App Password
- GitHub

## How to Run
1. Install dependencies:
   ```bash
   pip install requests
   ```

2. Run the program:
   ```bash
   python weather.py
   ```
